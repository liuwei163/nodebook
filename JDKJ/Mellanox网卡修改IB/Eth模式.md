
### 修改网卡模式的简单命令

```shell
apt update
apt-get install mft kernel-mft-modules
mst start
mst status
ip add   #确定修改哪个网口
mlxconfig -d /dev/mst/mt4123_pciconf0  q|grep LINK      #查看网口状态
mlxconfig -d /dev/mst/mt4123_pciconf0 set LINK_TYPE_P1=2       #P1=1 是IB模式；P1=2是以太网模式
y
sync
reboot
重启生效

```
